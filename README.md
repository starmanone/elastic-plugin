## Introduction
This is a project to implement useful functionality for elasticsearch.
The main idea is an app based on django and python that interface with elastic and does stuff.
Now it's just a concept but it could grow in the future
***

## Features
* Web interface
* Mail alert based on elastic query results
* Python gui modules for graphical data representation
